# Blog
## Como rodar o projeto
- Entrar em blog-composer e rodar ```docker-compose up -d``` para criar o banco
- Abrir blog-spring pelo IntelliJ e rodar como projeto do Spring ou pela linha de comando:
    ``mvn clean package``
    ``java -jar blog-0.0.1-SNAPSHOT.jar``
- Abrir pasta blog-angular com npm 16 ou superior e rodar:
    ``npm install``
    ``ng serve``
- Acessar localhost:4200 no browser.
